# Project-Modeling-Car-Insurance-Claim-Outcomes
Projet ML pour l’assurance auto visant à identifier la feature la plus prédictive d’un sinistre (outcome). Les données sont nettoyées, les colonnes catégorielles transformées, chaque feature testée avec un pipeline SimpleImputer + LogisticRegression, et la meilleure feature est stockée dans best_feature_df.
